## Возможности

- Возможность автоматически закрывать канал (чтобы никто не зашел)
- Множество настроек
- Полностью бесплатно

---

## 📦 Установка

Клонирования репозитория
```bash
git clone https://github.com/Crone720/Discord-AutoVoice.git
```



## Установка зависимостей и Сборка проекта

```bash
npm install
npm run build
```

## Запуск
```bash
npm start
```
